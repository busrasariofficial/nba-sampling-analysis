NBA Oyuncularının Sayı Ortalaması Tahmini
Bu çalışma, 2023-2024 NBA sezonu verilerini kullanarak oyuncuların maç başına sayı ortalamalarını tahmin etmek amacıyla yürütülmüştür.

Metodoloji
Projeyi hazırlarken iki farklı örnekleme yöntemini karşılaştırdım:

Tabakalı Rastgele Örnekleme (TRÖ): Oyuncular pozisyonlarına göre (Guard, Forward, Center) tabakalara ayrılmış ve Neyman Dağıtımı kullanılmıştır.

Basit Rastgele Örnekleme (BRÖ): Karşılaştırma amacıyla kitle yapısı dikkate alınmadan seçim yapılmıştır.

Önemli İstatistikler
Örneklem Hacmi (n): 107

TRÖ Varyansı: 0.1943

BRÖ Varyansı: 0.54

Sonuç: Tabakalı örnekleme yöntemi, varyansı ciddi oranda düşürmüştür.
